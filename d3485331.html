ubuntu安装containerd

发表于 2022-11-07 分类于 containerd 本文字数： 1.9k 阅读时长 ≈ 2 分钟

官方文档：containerd/getting-started.md at main · containerd/containerd · GitHub

安装containerd

下载地址：https://github.com/containerd/containerd/releases

将其提取到/usr/local

配置通过systemd启动containerd

cannot move ‘containerd.service’ to ‘/usr/local/lib/systemd/system/containerd.service’: No such file or directory

安装 runc

下载地址：https://github.com/opencontainers/runc/releases

并将其安装：runc.<ARCH> /usr/local/sbin/runc

二进制文件是静态构建的，应该适用于任何 Linux 发行版。

安装 CNI 插件

下载地址：Releases · containernetworking/plugins (github.com)

二进制文件是静态构建的，应该可以在任何 Linux 发行版上运行。
