运行本地容器

命令说明：

  • -name mysql：为容器指定名称

  • -p 3306:3306：将容器的3306端口映射到主机的3306端口

  • -e MYSQL_ROOT_PASSWORD=123456：初始化root用户的密码

  • -d 后台运行容器，并返回容器ID；

  • -v 在主机系统上的适当卷上创建数据目录，例如/my/own/datadir
